# Goldwear Jewelers

A modern, high-end editorial website for **Goldwear Jewelers** in Maple Grove, MN. Built with Next.js, Tailwind CSS, TypeScript, and React.

## Features

- **Luxury Aesthetics**: Magazine-style layouts, Cormorant Garamond serif headers, Tenor Sans body copy, and gold accents. Staggered alternating layouts with desaturated-to-color duotone transitions on hover.
- **Local SEO & Schema Markup**: Custom local SEO metadata tags on every route and integrated JSON-LD structured schema for `JewelryStore` and `LocalBusiness`.
- **Custom Design Showcase**: Staggered custom creation workflow detailing 3D modeling stages and designer lines (Allison-Kaufman, Stuller, Facet Barcelona, and Rego).
- **Comprehensive Services**: Landing pages for Jewelry Repair and Watch Services.
- **Reviews & Testimonials**: Editorial integration of verified client reviews from Google and Yelp.
- **Contact & Map**: Responsive contact form and Google Maps coordinates for the Grove Drive, Maple Grove location.

## Technology Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4, Vanilla CSS variables
- **Icons**: Custom SVG vectors
- **Language**: TypeScript

## Development

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or the assigned port) in your browser to view the application.
